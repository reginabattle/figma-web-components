A high-fidelity implementation of the Figma SDS Button and Icon Button components using Light DOM web component architecture.

The standard text button (<sds-button>) shown across all three variants (Primary, Neutral, Subtle), sizes (Medium, Small), and states. Standard pseudo-states (like hover, active, and focus) are demonstrated side-by-side using dedicated static attributes for direct visual audit.
