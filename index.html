Simple Design System (SDS) - Button Suite

A high-fidelity implementation of the Figma SDS Button library (Buttons, Danger Buttons, Icon Buttons, and Button Groups) using Light DOM Web Components and advanced Foundations design tokens.

1. Standard Buttons Board

Standard text buttons (<sds-button>) shown across three variants (Primary, Neutral, Subtle) and two sizes (Medium, Small).
